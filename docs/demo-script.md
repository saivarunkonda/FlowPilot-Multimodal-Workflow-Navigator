# FlowPilot Demo Script (<4 minutes)

## Intro (0:00–0:30)
- "Manual web workflows are repetitive and error-prone."
- "FlowPilot observes the UI with screenshots, understands intent using Gemini multimodal, and executes clicks/typing across apps."

## Live Demo (0:30–3:00)
1. Open Gmail and FlowPilot extension popup.
2. Enter instruction: "Log in and draft an email to my manager about today’s release."
3. Click **Capture + Plan**.
4. Show plan summary, confidence, and generated steps.
5. Click **Run Last Plan**.
6. Interrupt with: "Actually, CC my teammate too."
7. Re-run **Capture + Plan** with updated instruction and execute.
8. Show backend logs + Firestore record for workflow history.

## Outro (3:00–3:30)
- "FlowPilot saves time, reduces operational errors, and enables automation without API integrations."
- "It combines multimodal reasoning, Cloud Run orchestration, Cloud Functions execution, and Firestore memory."
